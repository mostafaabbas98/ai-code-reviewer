# AI Code Reviewer

AI Code Reviewer is a Next.js application that uses Google's Gemini AI to automatically review code snippets for readability, structure, and maintainability.

Simply paste your code into the interface and receive instant feedback with scores, issues, and actionable suggestions.

Built with Next.js, TypeScript, Tailwind CSS, and the Vercel AI SDK.

To get started, install dependencies, set your Google AI Studio API key in `.env.local`, and run the development server.

## Getting Started

```bash
pnpm install
pnpm dev
```

Set your `GOOGLE_GENERATIVE_AI_API_KEY` in `.env.local` and open [http://localhost:3000](http://localhost:3000).
